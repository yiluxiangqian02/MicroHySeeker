Model 1600 Series Battery Tester

 

 

The model 1600 series 8-channel potetniostat/galvanostat is designed for battery testing. It has eight independent channels for a maximum current at ±3A (depending on the cell loading) per channel. The potential range is ±5V. The sampling rate is fixed at 5 Hz. You can start a channel first and the other channels later. Any channel can be stopped at any time while other channels are still running.

 

There are two models in the model 1600 series. The CHI1600 will allow slow cyclic voltammetry and linear sweep voltammetry, amperometric i-t curve, chronopotentiometry for constant current battery charge/discharge, and open circuit potential versus time. The CHI1610 adds multi-potential steps and multi-current steps to the CHI1600.

 

The model 1600 series has a same user interface as our other instrument series. It therefore has nearly all software features and the very similar user interface as our other instruments. The data format is also compatible.

 

The major difference in user interface is that there are eight channel number (1-8) buttons and a All (channels) button on the toolbar. At any time, if you click a button for a channel number, it will display the data of this channel, you can read a data file in to this channel, or save the data of this channel to disk. You can also change technique, parameters of this channel. When you set parameters of this channel, you can also set same parameters for other channels that are not running experiment for convenience. The All button will display data for all channels. With All button pressed, you can click run, it will then run all channels that are not currently running (but one at a time). If you click Stop, it will stop all channels.

 

You can switch to a different channel or to all channels at any time, by pressing the toolbar channel button. Double clicking the data graph when all channels are selected, or when this particular channel is selected and running experiment, will toggle from all channels to single channel or vice versa. If a channel is selected but not running, you have to press the toolbar channel button to switch to other channel or to all channels.

 

The instrument regularly send data to the PC every 0.2 seconds. There is a watchdog in the CPU that if the program does not get back to sampling loop for more than 0.2 second, the instrument will reset as just turn the power on. Obviously, data sending is stopped. If the PC senses the data is not received for more than 0.2 second, it knows that something wrong with the instrument. Under this condition, the PC software will save all running channel data to binary data files using the binary data format that can be read back later. Since the instrument will recover to startup conditions due to watchdog function, the PC software will try to handshake with the instrument. If correct response is received, the PC will re-start run for all unfinished channels. for instance, if the instrument power is lost momentary, when the power is on again, the data before instrument halt are saved and the new experiments for all unfinished channels start.

 

The instrument can be controlled by any PC running Windows XP or higher. It is easy to install and use. The user interface uses the classic Microsoft design, so if you are familiar with Windows applications, you can likely use the software without referring at all to this user manual or the comprehensive, context-sensitive help files that are included with the instrument software. The commands, parameters, and options have been written using terminology that most chemists are familiar with. A customizable toolbar allows quick access to the most commonly used commands.

 

The instrument provides many powerful functions, such as straightforward file handling, extensive experimental control, flexible graphics, various data analyses, and efficient digital simulation. Additional features include color, legend and font selection, data interpolation, visual baseline correction, data point removal, visual data point modification, signal averaging, Fourier spectrum, and a convenient technique-specific electrochemical equation viewer.

 

This model series includes a USB port (default) and a serial port for data communication with the PC, with switching capabilities on the rear panel of the instrument. Firmware is stored in flash memory, and as a result, instrument firmware upgrades can be transferred electronically and installed immediately.

 

                                                  Specifications

Potentiostat:

  Potential Range:  ± 5 V

  Maximum current:  ± 2 A (depending on the cell load)

  Compliance Voltage: ±6.5 V

  Applied potential resolution: 0.01% of potential range

  Applied potential accuracy: ±1 mV, ±0.02% of scale

  Measured current range:  ±20 nA to ±3 A in 10 ranges

  Measured current resolution: 0.01% of current range, minimum 3 pA

  Current measurement accuracy: 1%

  Input bias current: < 100 pA

 

Galvanostat:

  Galvanostat applied current range: 30 nA – 3A

  Applied current accuracy: 100 pA ±1%

  Applied current resolution: 0.03% of applied current range

  Measured potential range: ±5 V

  Measured potential resolution: 0.015% of measured range

 

Electrometer:

  Reference electrode input impedance: 1e12 ohm

  Reference electrode input bias current: <= 100 pA @ 25°C

 

Experimental:

  CV and LSV scan rate:  0.000001 to 0.005 V/s

  Potential increment during scan:  0.1 mV

  i-t sample interval:  0.2 sec

  Chronopotentiometry sample interval:  0.2 sec

  Multi-Potential Step sample interval:  0.2 sec

  Multi-Current Step sample interval:  0.2 sec

  Open Circuit Potential - Time sample interval:  0.2 sec

 

Other:

  Flash memory for quick software update

  USB port for data communication

  Maximum data length: 256K-16384K selectable

  Instrument Dimensions: 17" (W) × 15" (D) × 6" (H)

  Weight: ~14 lb.
