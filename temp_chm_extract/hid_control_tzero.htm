Zero Time command (Control menu)

 

During an Amperometric i-t Curve experiment, the Zero Time command set the time to zero, throwing away all previous data points. This command can facilitate the time reading and reduce the number of unusable data points, and is particularly useful for electrochemical detection.

 

This command can only be used while an Amperometric i-t Curve experiment is in progress, and it cannot be used for other techniques.
