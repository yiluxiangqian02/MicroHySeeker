Help Topics command (Help menu)

 

Use this command to display the opening screen of help. From the opening screen, you can jump to step-by-step instructions for using the program and various types of reference information.

 

Once you open help, you can click the Home button whenever you want to return to the opening screen.

 

Context-sensitive help is available for the pull-down menus and the buttons on the toolbar. Press Shift+F1 to activate context-sensitive help (the mouse pointer will change to an arrow and question mark), and click on a menu command, toolbar button, or other part of the application window to learn more about it.
