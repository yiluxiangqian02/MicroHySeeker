CV Simulate command (Sim menu)

 

Use this command to generate simulated CV data.

 

The reaction mechanism, concentrations, kinetic parameters, and experimental parameters must be first set with the Mechanism command before the simulation is run.

 

In order to do simulation, make sure the instrument is correctly connected to your PC, e.g. by running the Self Test command. The program will check for the proper instrument model. It only needs to check once after the program is started. After the first fit, you do not need to have the instrument connected anymore.

 

You can copy the screen graphics to the clipboard during the simulation.
