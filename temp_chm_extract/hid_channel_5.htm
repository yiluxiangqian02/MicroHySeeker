Ch 5 command (Channel menu)

 

Use this command to select Channel 5.

 

Its toolbar button is:

 

At any time, if you click a button for a channel number, it will display the data of this channel, you can read a data file in to this channel, or save the data of this channel to disk. You can also change technique, parameters of this channel. When you set parameters of this channel, you can also set same parameters for other channels that are not running experiment for convenience.

 

You can switch to a different channel or to all channels at any time, by pressing the toolbar channel button. Double clicking the data graph when all channels are selected, or when this particular channel is selected and running experiment, will toggle from all channels to single channel or vice versa. If this channel is selected but not running, you have to press the toolbar channel button to switch to other channel or to all channels.

 

The maximum current for each channel is 3A. However, it may still exceed the maximum power dissipation of the power amplifier if most voltage drop occurs at the power amplifier. The power dissipation of the power amplifier can be calculated according to

(7.5V - voltage at counter electrode) * current. For instance, it the load is a 1 ohm resistor and current is 3A, the voltage drop at the load is 3V. The voltage drop at the power amplifier is 7.5V - 3V = 4.5V. The power dissipation on the power amplifier is 4.5V * 3A = 13.5W. This is about the maximum power allowed for the power amplifier. During experiment, the average power in 3 minutes (180 seconds) will be calculated. If it exceeds 13.5W, the experiment will automatic stop, in order to protect the power amplifier. To avoid this happen, one can insert a small power resistor at the counter electrode side to reduce the power amplifier voltage drop.
