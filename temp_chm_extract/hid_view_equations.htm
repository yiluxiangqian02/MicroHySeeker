Equations command (View menu)

 

This command opens the help browser to view equations related to the electrochemical technique currently invoked.

 

The following is a list of available equations. For more details about the equations and how to use them, please refer to "Electrochemical Methods", A.J. Bard and L.R. Faulkner, Wiley, New York, 1980 (1st ed), 2001 (2nd ed).
