Peak Definition dialog box (IT)

 

The following are available in the Peak Definition dialog box for the IT technique:       

 

Report Option

 

Report retention time, peak current, and/or peak area.

 

Minimum Half Peak Width

 

Peaks with half-width smaller than the specified value will be ignored; adjust peak width accordingly.

 

This parameter is only meaningful for automatic result reporting.

 

Maximum Half Peak Width

 

Peaks with half-width larger than the specified value will be ignored; adjust peak width accordingly.

 

This parameter is only meaningful for automatic result reporting.

 

Peak Current Threshold

 

Peaks with height smaller than the specified value will be ignored; adjust threshold accordingly.
