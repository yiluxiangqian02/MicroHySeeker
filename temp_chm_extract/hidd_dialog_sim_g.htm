Circuit Element Properties

 

Element

Impedance

Conductance

Phase

R (resistor)

ZR = R

YR = 1/R

φ = 0 (frequency-independent)

C (capacitor)

ZC = -j/ωC

YC = jωC

φ = π/2

L (inductor)

ZL = jωL

YL = -j/ωL

φ = -π/2

W (Warburg)

ZW = (jω)-1/2/Yo

YW = (jω)1/2 Yo

φ = π/4

Q (phase)

ZQ = (jω)-n/Yo

YQ = (jω)n Yo

φ = nπ/2, 0 < n < 1

CC (Cole-Cole)

ZCC = R / [1+(jωτ)α]

YCC = [1+(jωτ)α] / R

tan φ = (ωτ)αsin(απ/2) / [1+(ωτ)αcos(απ/2)]

CD (Cole-Davidson)

ZCD = R / (1+jωτ)α

YCD = (1+jωτ)α / R

tan (φ/α) = ωτ

FD (Finite Layer Diffusion)

ZFD = 1/YFD

YFD = Yo sqrt(jω)

tanh[δ sqrt(jω/D)]

tan (φ) = arg (YFD)

OFD (Open Finite Layer Diffusion)

ZOFD = 1/YOFD

YOFD = Yo sqrt(jω)

coth[δ sqrt(jω/D)]

tan (φ) = arg (YOFD)

GI (Gerischer Impedance)

ZGI = 1/YGI

YGI = Yo sqrt(k + jω)

tan (φ) = arg (YGI)
