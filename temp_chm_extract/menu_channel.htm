Channel menu commands

 

The Channel menu allows channel selection with the following commands:

 

Ch 1          Select Channel 1

Ch 2          Select Channel 2

Ch 3          Select Channel 3

Ch 4          Select Channel 4

Ch 5          Select Channel 5

Ch 6          Select Channel 6

Ch 7          Select Channel 7

Ch 8          Select Channel 8

All Ch          Select All Channels

 

The above commands have corresponding toolbar buttons for quick access: 1-8 buttons and a All (channels) button on the toolbar. At any time, if you click a button for a channel number, it will display the data of this channel, you can read a data file in to this channel, or save the data of this channel to disk. You can also change technique, parameters of this channel. When you set parameters of this channel, you can also set same parameters for other channels that are not running experiment for convenience. The All button will display data for all channels. Obviously you can not change techniques or parameters at this state.

 

With All button is pressed, you can click run, it will then run all channels that are not currently running (but one at a time). If you click Stop, it will stop all channels.

 

You can switch to a different channel or to all channels at any time, by pressing the toolbar channel button. Double clicking the data graph when all channels are selected, or when this particular channel is selected and running experiment, will toggle from all channels to single channel or vice versa. If a channel is selected but not running, you have to press the toolbar channel button to switch to other channel or to all channels.

 

The maximum current for each channel is 3A. However, it may still exceed the maximum power dissipation of the power amplifier if most voltage drop occurs at the power amplifier. The power dissipation of the power amplifier can be calculated according to

(7.5V - voltage at counter electrode) * current. For instance, it the load is a 1 ohm resistor and current is 3A, the voltage drop at the load is 3V. The voltage drop at the power amplifier is 7.5V - 3V = 4.5V. The power dissipation on the power amplifier is 4.5V * 3A = 13.5W. This is about the maximum power allowed for the power amplifier. During experiment, the average power in 3 minutes (180 seconds) will be calculated. If it exceeds 13.5W, the experiment will automatic stop, in order to protect the power amplifier. To avoid this happen, one can insert a small power resistor at the counter electrode side to reduce the power amplifier voltage drop.
