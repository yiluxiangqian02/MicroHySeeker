List Data File command (File menu)

 

This command opens the List Data File dialog box.

 

Use this command to display a saved data file in text mode. Formatting options are available in the Text File Format command.

 

The current data remains unchanged when this command is executed. To view the current data similarly, use the Data Listing command under View menu.

 

The List Data File command presents this dialog box.

 

The following options allow you to specify the name and location of the file you're about to display in text mode:

 

File name

 

Type or select the filename you want to display. The box above lists files with the extension you select in the "Files of type" box.

 

You don't have to type the extension.  The system will automatically attach an extension of ".bin" to the filename. Other extensions are not allowed.

 

Files of type

 

Select the type of file you want to list. Only ".bin" files (binary data) are available.
