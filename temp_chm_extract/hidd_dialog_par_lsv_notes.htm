Notes

  Init E and Final E should be at least 0.01 V apart.

  The maximum potential scan range is 13.1 V.

  When the scan rate is high, the data sampling interval will be automatically increased.

  When the scan rate is below 0.01 V/s, the sensitivity scale during the run can be automatically adjusted according to the current level. When Auto Sens is enabled, the value of the Sensitivity field will have no effect on the measurement. However, the smallest available automatic sensitivity will be 1e-6 A/V instead of 1e-12 A/V. The Picoamp Booster will not work either. In order to specify smaller sensitivities, the automatic sensitivity switching option must be disabled.

  Linear polarization resistance plots can be generated using the Special Plots command under the Graphics menu.
