Chronopotentiometry Parameters dialog box

 

In Chronopotentiometry (CP), two current levels can be specified to pass through the working electrode, with the switching between the two dictated by time or potential. Potential is recorded as a function of time. The following diagram shows the current waveform applied as a function of time.

Notes

  Discharge (cathodic) and charge (anodic) currents correspond to reduction and oxidation, respectively. When the Low E limit is reached during reduction, the current polarity will automatically be switched to anodic. Similarly, when the High E Limit is reached during oxidation, current will automatically be switched to cathodic. When the number of current polarity switches (Segments) is reached, the experiment stops.

  The Initial Polarity parameter specifies the initial current polarity.

  During the experiment, data update and storage rates will be the same (i.e., dictated by the Data Storage Intvl parameter).

  The data sampling interval should be chosen according to the length of the experiment. The longer the experiment, the longer the sampling interval should be.

  During an experiment, whenever the data exceed the maximum allowed points, the data storage interval will be doubled automatically. Therefore, data points will not overflow for an unexpectedly long experiment.

  A large number of segments can be specified, but data will only be stored for the first 400 segments. Later segments will be displayed during the run but not stored.

  The current polarity can be switched when either the specified potential or the specified time has been reached. The switching time can be different for cathodic and anodic switching. Note, however, that even with Time switching selected, if the limiting potential is reached, the current polarity will still be reversed in order to protect the electrode.

  Instrument can store the running data to the hard drive, if "Save retrieve data during run" option in the System command under the Setup menu is checked.  If an experiment terminated unexpectedly, the data will be saved. when you start the program, it will automatic load it. All 8 channel data will be retrieved. You need to save at that time. Otherwise if you click run, the data will be overridden by the next running experimental data.

  Data can be automatically saved at the end of experiment, if "Auto Data Save After Run" option in the System command under the Setup menu is checked.  The file name saved will look like this: 20160102_142345_CH5_CV.bin. The first part is the date (1/2/2016), the second part is time (14:23:45), the third part is the channel number (channel 5), the 4th part is the technique.
