Amperometric i-t Curve dialog box

 

In the Amperometric i-t Curve technique (i-t), a constant potential is applied, and current is recorded as a function of time. The following diagram shows the potential waveform applied as a function of time and the current sampling scheme.

Notes

  During an experiment, whenever the data exceed the maximum allowed points, the data storage interval will be doubled automatically. Therefore, data points will not overflow for an unexpectedly long experiment.

   Instrument can store the running data to the hard drive, if "Save retrieve data during run" option in the System command under the Setup menu is checked.  If an experiment terminated unexpectedly, the data will be saved. when you start the program, it will automatic load it. All 8 channel data will be retrieved. You need to save at that time. Otherwise if you click run, the data will be overridden by the next running experimental data.

  Data can be automatically saved at the end of experiment, if "Auto Data Save After Run" option in the System command under the Setup menu is checked.  The file name saved will look like this: 20160102_142345_CH5_CV.bin. The first part is the date (1/2/2016), the second part is time (14:23:45), the third part is the channel number (channel 5), the 4th part is the technique.
