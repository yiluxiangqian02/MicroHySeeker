Zoom In command (Graphics menu)

 

Use this command to magnify the data plot.

 

This command has a toolbar button:

 

To enter zoom mode, either click on the toolbar button or on its entry in the Graphics menu. A check mark appears next to the menu item when zoom mode is enabled, and its toolbar button will be depressed.

 

Once in zoom mode, the mouse cursor will appear as an up arrow in the data plot area. To zoom in, left-click and drag between diagonal corners of the box containing the area you want to view more closely. Release the mouse button to zoom in.

 

To zoom back out and exit zoom mode, click the Zoom In command or toolbar button again.
