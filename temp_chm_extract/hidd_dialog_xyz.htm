Probe Status dialog box

 

The SECM Probe Status command presents this dialog box.

 

The relative position of any of the axes can be reset to zero at any time by clicking the corresponding "Zero" button. If the probe and/or substrate electrode is turned on when the probe is scanning, the corresponding current(s) will also be displayed. Position and current are refreshed constantly.

 

This dialog box is modeless and can remain open to allow you to monitor the probe status at all times.
